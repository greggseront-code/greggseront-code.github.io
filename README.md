# AI-Augmented Software Engineering - Slides

This repository contains the Slidev source for the presentation "AI-Augmented Software Engineering".

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher recommended)

### Installation

Install the dependencies:

```bash
npm install
```

### Usage

#### Start Development Server
To preview the slides interactively in your browser:

```bash
npm run dev
```

#### Export to PDF
To generate a PDF version of the presentation:

```bash
npm run export
```

#### Build for Production
To build a static version for hosting:

```bash
npm run build
```

## 🛠 Tech Stack

- [Slidev](https://sli.dev/) - Presentation tool for developers
- [UnoCSS](https://unocss.dev/) - Instant on-demand atomic CSS engine
- [Playwright](https://playwright.dev/) - Used for PDF rendering
